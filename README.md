# .dotfiles

## Supported Shells for Aliases
* Bash
* Fish
* PS7

## How to use
1) ```cd $HOME```
2) ```git clone https://github.com/dubskysteam/.dotfiles```
3) ```cd .dotfiles```
4) ```sudo chmod +x ./Install.sh```
5) ```sudo ./Install.sh```

### NeoVim
   1) ```:source %```
   2) ```:PlugInstall```
